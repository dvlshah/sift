# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1, as its code of conduct.

In short: be respectful and constructive. Treat everyone with courtesy in issues, pull requests, reviews, and every other project space. Unwelcome conduct — harassment, personal attacks, or demeaning comments — is not tolerated.

## Scope

This applies within all project spaces and when an individual is representing the project in public.

## Reporting

To report a concern, use this repository's private reporting on GitHub (**Security → Report a vulnerability** also reaches the maintainers), or open an issue for non-sensitive matters. Maintainers will review and respond in good faith, and will respect the privacy of any reporter.

The full text and enforcement guidelines are available in the Contributor Covenant v2.1 linked above.
